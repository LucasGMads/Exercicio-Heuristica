# Projeto IDA* - Mapa de Cidades

## Descrição
Implementação do algoritmo IDA* para encontrar o melhor caminho entre cidades.

## Funcionalidades
- Escolha de origem e destino
- Suporte a múltiplos destinos
- Cálculo do custo total
- Visualização gráfica do mapa

## Tecnologias
- Python
- NetworkX
- Matplotlib

## Como executar
```bash
pip install networkx matplotlib
python main.py
